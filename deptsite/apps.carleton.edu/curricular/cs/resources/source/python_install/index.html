Installing Python 2.x On Your Home Computer

Windows XP and Vista and Windows 7

Download and Install Python 2.7.1:

  1. Go to www.python.org/download/releases/2.7.1
  2. Scroll down and pick Windows x86 MSI Installer (2.7.1).  Do NOT install the 64 bit version;  the 64 bit version conflicts with PIL, a library used in CS 111.
  3. Save the installer. When it is downloaded, run it.
  4. On the first installer screen select "install for all users". Click next.
  5. Select where you want python to be installed. If the default is fine, click next.
  6. In the Customization screen, click on the python drop down box at the top of the tree, and select "Entire feature will be installed on local hard-drive". Click next.
  7. Click finish.

To test your installation:

  1. Click start -> programs
  2. Click Python 2.7 -> IDLE. A python shell should open up
  3. At the prompt type "print 'hello world'"
  4. "hello world" should be echoed back.

Add python to your path:

  1. Right click on my computer, and pull down to properties.
  2. Skip this step if you are not running Windows XP. Click on "Advanced system settings" on the left "Tasks" bar.
  3. Click on Advanced.
  4. Click on Environment Variables
  5. In the "system variables" list, select "Path", and click on Edit.
  6. Add ";c:\Python27" to the end of the "Variable value:" field. Note: DO NOT ADD the double quotes
  7. Click "ok" three times.

To test the path:

  1. Click Start/All Programs/Accessories/Command Prompt.
  2. Type "python" without the double quotes.
  3. You should see a python prompt; try entering "1+1", if you get "2" then everything is running correctly.

If you are looking for a windows editor that has more functionality than notepad and still saves files as text only, you might want to try ConTEXTor emacs.

Mac OS

Most modern Mac operating systems come with python preinstalled.

To access the terminal:

  1. Go to Applications ->Utilities.
  2. Terminal should be in that list.
  3. If you want permanent access to the terminal just drag it to the dock.

To test your installation of python:

  1. Open a terminal.
  2. Type python
  3. A prompt should show up that looks like ">>>"

If you are looking for a text editor, there is a lot of personal preference involved, two freeware versions for the mac are TextWrangler and SubEthaEdit. If you don't want to download a new editor, a standard simple text editor that comes with most Macs is TextEdit.

To use TextEdit as your default editor:

  1. Open TextEdit (in your applications folder)
  2. Click TextEdit -> Preferences
  3. Change the format to plaintext
  4. Click on the tabe marked "Open and Save"
  5. Uncheck the box "Add .txt to plain text files"
  6. Open a new document and you should be ready to go, just make sure when you save to save as "YourProgramName.py"

Linux

Most versions of linux come with python already installed. If yours doesn't or if you just can't find it, contact Mike Tie (mtie@carleton.edu).

You will have lots of choices of editors under linux, a simple one that we have had good luck with is NEdit.

Problems:

If you run into trouble, please contact Mike Tie (mtie@carleton.edu).
